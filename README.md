# cryptomining

Simple script that helps you find crypto wallet. This is just a simulation.


Set up for Termux
```
pkg install git
pkg install python3
git clone https://github.com/yuhertz/cryptomining
cd cryptomining
python3 mine.py
```

Set up for Kali Linux
```
sudo apt install git
sudo apt install python3
git clone https://github.com/yuhertz/cryptomining
cd cryptomining
python3 mine.py
```
